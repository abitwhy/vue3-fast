    ellipsis

    @mixin ellipsis($lines: 1) { ... }

    以省略号 ... 标记的溢出隐藏。

    元素必须有宽度/高度

    flex 布局通过 flex: 1 设置的元素也是有宽度/高度的。 值得一提的是，flex: 1 等价于 flex: 1 1 0 而非 flex: 1 1 auto。 参见知乎相关讨论。

    parameter Nameparameter Descriptionparameter Typeparameter Default value

    setFont

    @mixin setFont($fonts) { ... }

    Description

    设置字体

    并帮你做一些优化。

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $fonts

    字体集合

    		Map(attr-name: typeof url)— none

    Example

    基本用法

    .somewhere {
  @include setFont(
    (
      font-name: url(path/to/font),
    )
  );
}

    混入内容

    .somewhere {
  @include setFont(
    (
      font-name: url(path/to/font),
    )
  ) {
    // 自定义混入内容
    // 如：所需加载的字符编码范围
    unicode-range: U+30-39, U+61-7a, U+41-5a;
  }
}

    Output

    基本用法
    .somewhere {
  @font-face {
    font-family: font-name;
    src: local(font-name), url(path/to/font);
    font-display: swap;
    -webkit-font-smoothing: antialiased;
    -moz-osx-font-smoothing: grayscale;
  }
}
    混入内容
    .somewhere {
  @font-face {
    font-family: font-name;
    src: local(font-name), url(path/to/font);
    font-display: swap;
    -webkit-font-smoothing: antialiased;
    -moz-osx-font-smoothing: grayscale;
    unicode-range: U+30-39, U+61-7a, U+41-5a;
  }
}
